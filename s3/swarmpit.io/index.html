Features

Swarmpit provides nice & clean way to manage your Docker Swarm cluster.

Stack
management

Compose new stack manually or let Swarmpit generate one from application state.

Resource
monitoring

Displays information about the use of hardware (CPU, memory, disk) in real-time

Service
management

Deploy and manage your services

Smart
search

Search for images accross public and private registries

Shared
access

Allow multiple users to manage your Docker Swarm cluster safely

Private
registry

Pull private repositories from Docker Hub or custom registries

Service
autoredeploy

Checks whether new service image has been published and update service accordingly

Now we're succesfully running Docker on premises, 4 major fintech clusters in Azure and finally our internal infrastructure in AWS which was moved from Docker Cloud by using Terraform template. We believe that this strong Docker experience is going to make your life easier.
