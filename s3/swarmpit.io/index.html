Features

Swarmpit provides nice & clean way to manage your Docker Swarm cluster.

Stack management

Compose new stack manually or let Swarmpit generate one from application state. It’s never been easier

Service autoredeploy

Checks whether new service image has been published and update service automatically

How to install

git clone https://github.com/swarmpit/swarmpit
docker stack deploy -c swarmpit/docker-compose.yml swarmpit

app is published on port 888 by default and you can sign in with user/pass: admin/admin
