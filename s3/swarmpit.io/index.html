Features

Swarmpit provides nice &and; clean way to manage your Docker Swarm cluster.

Service management

Deploy and manage your services


Smart search

Search for images accross public and private registries


Shared access

Allow multiple users to manage your Docker Swarm cluster safely

Private registry

Pull private repositories from Docker Hub or custom registries

Service autoredeploy

Checks whether new service image has been published and update service automatically

How to install

git clone https://github.com/swarmpit/swarmpit -b 1.1
docker stack deploy -c swarmpit/docker-compose.yml swarmpit
